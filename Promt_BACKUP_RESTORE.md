# Prompt BACKUP_RESTORE

## 頁面目的
建立 Backup / Restore 頁面，用於匯出與匯入系統設定。

## 生成要求
- 分成 Export Configuration 與 Import Package
- 要有操作區與說明區
- 適合維運與備援情境

## 內容區塊
- 頁面標題：Backup / Restore
- Export Configuration
  - export scope
  - export button
  - summary
- Import Package
  - file input
  - validation notes
  - restore action
  - rollback / warning text

## 風格限制
- 穩重、清楚
- 重點在操作與風險提示
