# Prompt MATERIAL DARK

## Objective
Create a single dark-mode material board for the CYP project.

This board is not a marketing page and not an app dashboard. It is a dark-theme design-system and component-material reference board for Stitch generation and design handoff.

This version must include:
- brand identity
- product tone
- full dark-mode color palette
- reusable UI materials
- pad preview materials
- template layout and button template materials

Use the CYP control-console style in dark mode:
- dark surfaces
- controlled cyan-blue primary accents
- clear contrast for operational readability
- enterprise-grade tone
- no decorative hero or presentation-slide styling

## Dark Mode Rules

### DO
- preserve the CYP cyan-blue identity in active and primary states
- keep strong contrast for text, states, and controls
- make surfaces feel layered, not flat black
- use yellow highlight carefully for selected pad emphasis
- keep the interface technical, calm, and operational

### DON'T
- do not make the UI neon or gaming-themed
- do not use oversized gradients as decorative backgrounds
- do not blur hierarchy between panel, canvas, and overlay levels
- do not reduce component readability for aesthetic effect
- do not introduce marketing copy or slide-like storytelling

---

## Output Goal

The final screen should read as:
- a dark-mode CYP design-system board
- a component material board for future Stitch generation
- a clear handoff reference for dark-theme CYP screens

This board should function as the dark companion to the light material board and should be suitable for generating dark-mode CYP screens consistently.
