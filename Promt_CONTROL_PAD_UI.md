# Prompt CONTROL_PAD_UI

## 頁面目的
建立控制 Pad 的 runtime UI 頁面，用來模擬實際牆面面板或平板控制介面。

## 生成要求
- 需呈現可操作的 Pad 畫面
- 以多頁控制按鈕為主
- 顯示系統 feedback 區塊
- 適合會議室、影音或環控情境

## 內容區塊
- 頁面標題：Control Pad UI
- Control Pages 區塊
- 多頁按鈕網格
- System Feedback 區塊
- 狀態提示、模式切換、訊號顯示

## 按鈕類型
- toggle
- scene select
- source select
- volume
- power
- slider / dimmer

## 風格限制
- 要像真實控制面板
- 不要做成一般網站內容頁
