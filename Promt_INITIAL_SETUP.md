# Prompt INITIAL_SETUP

## 頁面目的
建立 Initial Setup 頁面，作為首次開機或 reset 後的設定導引頁。

## 生成要求
- 要有 wizard 感，但仍維持單頁概覽
- 呈現首次設定步驟與 recovery flow
- 可以是骨架頁，但不能空泛

## 內容區塊
- 頁面標題：Initial Setup
- first boot guide
- setup steps
  - network
  - device naming
  - language
  - admin password
- recovery / factory reset notes
- next step CTA

## 風格限制
- 清楚引導
- 適合設備首次啟用流程
