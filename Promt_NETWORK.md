# Prompt NETWORK

## 頁面目的
建立 Network 頁面，管理 Ethernet 與 Wi-Fi 連線設定。

## 生成要求
- 分為 Wired Network 與 Wireless Networks
- 同時呈現設定區與狀態區
- 要像真實裝置網路設定頁

## 內容區塊
- 頁面標題：Network
- Wired Network
  - DHCP / Static
  - IP
  - Mask
  - Gateway
  - DNS
- Wireless Networks
  - Wi-Fi toggle
  - SSID list
  - password modal 入口
  - connected status
- network summary

## 風格限制
- 清楚顯示目前連線狀態
- 企業設備設定風格
