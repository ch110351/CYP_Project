# Prompt SCHEDULING_LIST

## 頁面目的
建立 Scheduling 列表頁，集中管理所有時間排程。

## 生成要求
- 以 schedule table 為主
- 支援搜尋與 trigger type filter
- 顯示 rule summary 與 next run

## 內容區塊
- 頁面標題：Scheduling
- Search
- Filter
  - All
  - Interval
  - Recurring
  - Calendar
- table columns
  - Schedule Name
  - Type
  - Bound Item
  - Trigger Type
  - Rule Summary
  - Next Run
  - Status
  - Actions
- Create Schedule CTA

## 風格限制
- 專業排程後台
- 讓資料關係一眼可懂
