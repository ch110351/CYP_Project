# Prompt ADMIN

## 頁面目的
建立 Admin 頁面，提供安全設定與系統維護操作。

## 生成要求
- 分為 Access Control 與 System Actions
- 需同時包含安全表單與維護操作區
- 適合設備管理員使用

## 內容區塊
- 頁面標題：Admin
- Access Control
  - admin password
  - password confirm
  - access role
- System Actions
  - import configuration
  - export configuration
  - download log
  - OTA update
  - manual firmware upload
  - reboot
  - factory reset
- confirm action dialog

## 風格限制
- 穩重正式
- 高風險操作要明確區隔
