# Prompt DEVICE_DISCOVERY

## 頁面目的
建立 Device Discovery 頁面，作為搜尋區網裝置與檢視掃描結果的工程工具頁。

## 生成要求
- 以掃描結果 table 為主
- 顯示 reachability、protocol 與基本設備資訊
- 需有 start discovery 的入口

## 內容區塊
- 頁面標題：Device Discovery
- discovery action bar
- results table
- columns
  - Device Name
  - IP
  - Protocol
  - Reachability
  - Status
  - Actions
- detail review modal

## 風格限制
- 工程工具頁
- 資訊清楚、可掃描
