# Prompt DATE_TIME

## 頁面目的
建立 Date and Time 頁面，處理時間來源與時間格式設定。

## 生成要求
- 分為 Time Source 與 Current Display
- 同時顯示設定表單與結果預覽
- 支援 NTP / Manual 與 12 / 24 格式

## 內容區塊
- 頁面標題：Date and Time
- Time Source
  - NTP
  - Manual
  - date input
  - time input
- Current Display
  - current date
  - current time
  - 12/24 hour preview

## 風格限制
- 表單簡潔
- 預覽要明確
