# Prompt MACRO_MANAGEMENT

## 頁面目的
建立 Macro Management 頁面，管理所有 macro 流程。

## 生成要求
- 需有搜尋、列表、分頁、CTA
- 可清楚分辨 macro 名稱、描述、狀態與動作
- 支援空狀態與建立第一筆流程的提示

## 內容區塊
- 頁面標題：Macro Management
- 搜尋欄
- macro summary
- macro list table
- actions
  - edit
  - duplicate
  - test
  - delete
- create macro button
- empty state

## 風格限制
- 後台管理樣式
- 以流程管理感為主
