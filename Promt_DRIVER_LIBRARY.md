# Prompt DRIVER_LIBRARY

## 頁面目的
建立 Driver Library 頁面，供使用者搜尋、篩選與檢視 driver 資料。

## 生成要求
- 採 master-detail 介面
- 左側品牌、中間型號、右側 driver detail
- 提供 import / export / create device 動作

## 內容區塊
- 頁面標題：Driver Library
- Search / Filter toolbar
- Brands
- Models
- Driver detail
  - protocol
  - commands
  - capability
  - notes
- actions
  - import driver
  - export driver
  - create device

## 風格限制
- 專業工程工具頁
- 信息密集但結構清楚
