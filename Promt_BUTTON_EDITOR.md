# Prompt BUTTON_EDITOR

## 頁面目的
建立 Button Editor 頁面，用於設定單一控制按鈕的屬性與行為。

## 生成要求
- 偏設定頁
- 明確區分 Button Properties 與 Selected Button 預覽
- 支援 label、icon、binding、execute behavior

## 內容區塊
- 頁面標題：Button Editor
- Button Properties
  - label
  - icon
  - command / macro binding
  - execution behavior
    - Normal
    - Toggle
    - Repeat
- Selected Button
  - preview
  - summary
  - current mapping
