# Prompt DEVICE_CREATE

## 頁面目的
建立 Add Device 表單頁，用於從 driver 或設定資料建立新裝置。

## 生成要求
- 需分為 Device Setup 與 Summary
- 讓使用者看到設定完成後的能力預覽
- 表單偏工程導向

## 內容區塊
- 頁面標題：Add Device
- Device Setup
  - device name
  - vendor / model
  - protocol
  - IP
  - Port
  - credentials / notes 保留欄位
- Summary
  - selected driver
  - connection summary
  - capability preview

## 風格限制
- 清楚分欄
- 可讀性高
