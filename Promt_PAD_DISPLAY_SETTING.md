# Prompt PAD_DISPLAY_SETTING

## 頁面目的
建立 Pad Display Setting 頁面，管理顯示、亮度、主題與螢幕行為。

## 生成要求
- 分為 Display Setup 與 Current State
- 需要顯示偏好設定與即時摘要
- 適合平板或控制面板裝置設定

## 內容區塊
- 頁面標題：Pad Display Setting
- Display Setup
  - brightness
  - theme mode
  - pad language
  - screen lock
  - idle minutes
  - screen saver content
- Current State
  - brightness badge
  - theme badge
  - lock status

## 風格限制
- 設定明確、摘要清楚
