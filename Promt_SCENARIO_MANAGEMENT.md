# Prompt SCENARIO_MANAGEMENT

## 頁面目的
建立 Scenario Management 頁面，作為面板場景、按鈕配置與頁面版面的主要編輯中心。

## 生成要求
- 以 split-view / builder 方式呈現
- 要像可編輯的 panel designer
- 支援頁面切換、button template、layout template、preview canvas

## 內容區塊
- 頁面標題：Scenario Management
- Scenarios 區
- Meeting Mode 或選定場景摘要
- 左側
  - layout templates
  - button templates
  - page tabs
- 中間
  - control pad preview
  - grid canvas
- 右側或 modal
  - selected button config
  - action binding
  - icon picker

## 互動提示
- drag and drop
- multi-page
- unsaved changes warning
- layout compatibility confirm

## 風格限制
- 要有產品級 editor 感
