# Prompt DIALOGS_SHOWCASE

## 頁面目的
建立 Dialogs / Modals 展示頁，整理專案共用的對話框樣式。

## 生成要求
- 需收納多種 modal 範例
- 用於設計與前端對齊，不是 runtime 功能主頁
- 保持 component showcase 形式

## 內容區塊
- 頁面標題：Dialogs / Modals
- Device Review
- Delete Schedule
- Import Driver
- 每個 dialog 顯示：
  - header
  - body
  - actions

## 風格限制
- 以元件展示方式排列
- 保持整體品牌一致
