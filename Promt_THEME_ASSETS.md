# Prompt THEME_ASSETS

## 頁面目的
建立 Theme and Assets 頁面，展示主題、wallpaper 與 icon 資產管理方向。

## 生成要求
- 分成 Visual Presets 與 Wallpaper / Icons
- 要兼具參考頁與管理頁感
- 支援 theme 資訊卡與資產格狀呈現

## 內容區塊
- 頁面標題：Theme and Assets
- Visual Presets
  - theme cards
  - light / dark / branded presets
- Wallpaper / Icons
  - icon summary
  - wallpaper previews
  - asset management area

## 風格限制
- 保持企業產品風格
- 不要做成純設計展示頁
