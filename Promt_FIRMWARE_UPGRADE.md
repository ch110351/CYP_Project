# Prompt FIRMWARE_UPGRADE

## 頁面目的
建立 Firmware Upgrade 頁面，支援 OTA 與本地套件升級。

## 生成要求
- 分為 Online Upgrade 與 Manual Package
- 需顯示版本、套件與操作入口
- 要有維運安全感

## 內容區塊
- 頁面標題：Firmware Upgrade
- Online Upgrade
  - current version
  - latest version
  - OTA button
- Manual Package
  - file input
  - package validation notes
  - upgrade button

## 風格限制
- 不要做成下載中心
- 要像設備升級頁
