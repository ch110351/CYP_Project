# Prompt HOME

## 頁面目的
建立控制系統平台的首頁 Dashboard，作為進入系統後的第一個管理畫面。

## 生成要求
- 這不是 marketing 首頁，而是 operational dashboard
- 需呈現 Pad / Device 概況與控制預覽
- 要讓使用者一進入就知道設備是否在線、目前版本與控制介面長相

## 內容區塊
- 頁面標題：Home
- Device Overview / Pad Information 卡片
  - Device Name
  - Model
  - MAC Address
  - IP Address
  - Firmware Version
  - Network
- Online 狀態 badge
- Control Pad 預覽卡片
- 多頁控制按鈕預覽
- 時間、Wi-Fi、現場控制情境資訊

## 互動提示
- 支援控制頁切換
- 按鈕需有 on / selected / alert 等狀態
- 可包含 slider 類型控制

## 風格限制
- 企業控制平台
- 淺色背景、白色面板
- 不要 hero banner
