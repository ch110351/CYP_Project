# Prompt MACRO_EDITOR

## 頁面目的
建立 Macro Editor 三欄式編輯頁，讓使用者以流程方式組裝 macro。

## 生成要求
- 採三欄 layout
  - Insert Actions
  - Flow Editor
  - Step Properties
- 支援拖拉與插入步驟
- 看起來像工程流程編排器

## 內容區塊
- 頁面標題：Macro Editor
- Insert Actions
  - Execute Command
  - Delay
  - IF / ELSE
  - Start Polling
  - Update Variable
  - Run Macro
- Flow Editor
  - step cards
  - connectors
  - drop zones
- Step Properties
  - selected step form
  - parameter editing
  - validation hints
- Save / Test actions

## 風格限制
- 不能太像一般表單頁
- 要有流程編輯器感
