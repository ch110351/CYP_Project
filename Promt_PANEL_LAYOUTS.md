# Prompt PANEL_LAYOUTS

## 頁面目的
建立 Template Layout Selection 頁面，提供控制面板格局選擇。

## 生成要求
- 需展示不同 grid/layout template
- 每個版型需可看出容量與結構差異
- 適合作為場景設計前置頁

## 內容區塊
- 頁面標題：Template Layout Selection
- layout gallery
- 每個 layout card 顯示
  - layout name
  - rows / cols
  - capacity
  - preview grid
- selected state

## 風格限制
- 視覺上清楚對比版型
- 不需太多文字
