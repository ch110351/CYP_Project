# Prompt INDEX

## 頁面目的
建立 `ui-static/index.html` 入口頁，作為整個 UI handoff package 的總覽與導覽首頁。

## 生成要求
- 建立一個 design handoff 入口頁，不是產品 runtime 首頁
- 需能快速列出所有主要頁面分類與連結
- 需顯示專案名稱、說明、交付目的與檔案分區
- 視覺上要延續企業控制平台風格

## 內容區塊
- 專案標題區
- handoff 說明
- sitemap 導覽清單
- 依群組分區：
  - Home
  - Control
  - Device
  - Scenes
  - Setting
  - Developer Tools
  - UI Reference

## 風格限制
- 不要做成行銷 landing page
- 用清楚卡片或清單呈現頁面索引
- 文字偏交付說明與頁面導覽
