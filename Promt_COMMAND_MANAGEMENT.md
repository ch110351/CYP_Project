# Prompt COMMAND_MANAGEMENT

## 頁面目的
建立 Command Management 頁面，管理所有裝置控制指令。

## 生成要求
- 以 table-based library 方式呈現
- 提供搜尋、摘要、分頁與操作按鈕
- 適合工程人員檢視與維護命令資料

## 內容區塊
- 頁面標題：Command Management
- 搜尋欄與 toolbar
- command summary
- command table
- table action buttons
  - edit
  - duplicate
  - test
  - delete
- create command primary CTA

## 欄位方向
- ID
- Name
- Interface
- Device
- Command Data
- Updated State / Status

## 風格限制
- 控制台後台風格
- 表格與操作性優先
