# Prompt SYSTEM_SETTINGS

## 頁面目的
建立 System Settings 頁面，管理登入、逾時與安全策略摘要。

## 生成要求
- 分成 Login / Timeout 與 Security Summary
- 偏管理設定頁
- 需有表單 + 摘要卡

## 內容區塊
- 頁面標題：System Settings
- Login / Timeout
  - login state
  - session timeout
- Security Summary
  - access control policy
  - role summary
  - session behavior

## 風格限制
- 清楚、正式
- 適合企業後台
