# Prompt STATES_SHOWCASE

## 頁面目的
建立 States Showcase 頁面，展示系統中的空狀態、搜尋無結果、刪除確認與 toast 樣式。

## 生成要求
- 作為 UI reference 頁
- 一頁整合多種 state pattern
- 適合設計 handoff 與前端實作對照

## 內容區塊
- 頁面標題：States Showcase
- Empty State
- Search Empty
- Delete Confirm
- Toast States
  - success
  - warning
  - error
  - info

## 風格限制
- 以 pattern library 呈現
- 各狀態要能明顯區分
