# Prompt DEBUG_CONSOLE

## 頁面目的
建立 Debug Console 頁面，提供事件記錄與命令測試的工程調試介面。

## 生成要求
- 需有 console output 區
- 可輸入 debug command
- 可顯示 diagnostic KPI cards

## 內容區塊
- 頁面標題：Debug Console
- debug metrics / KPI cards
- console output area
- command input form
- hint / helper area

## 風格限制
- 要像工程除錯頁
- 以可讀性為主
