# Prompt DEVICES_LIST

## 頁面目的
建立 Devices 列表頁，管理所有已建立的設備實例。

## 生成要求
- 以 table 呈現 device inventory
- 要能區分 driver-based 與 manual device
- 提供建立、檢視、編輯、刪除操作

## 內容區塊
- 頁面標題：Devices
- toolbar
- device table
- columns
  - Device Name
  - Type
  - Driver / Manual
  - Protocol
  - IP / Port
  - Status
  - Actions
- create device CTA

## 風格限制
- 工程設備管理後台
- 清楚反映裝置連接資訊
